Introduction

CSS (Cascading Style Sheets) is used to style HTML elements. It controls layout, color, fonts, and more.

CSS Selectors

Selectors allow you to target HTML elements to apply styles.

  • Element selector: h1 { color: blue; }
  • ID selector: #header { font-size: 24px; }
  • Class selector: .button { background-color: green; }

CSS Properties

Common properties include:

  • color, background-color
  • font-size, font-family
  • margin, padding, border

Types of CSS

CSS can be applied in three ways:

  • Inline: style="color:red;"
  • Embedded: inside <style> tag
  • External: separate .css file linked

Media-dependent Cascades

Use media queries to make your website responsive:

@media screen and (max-width: 768px) {
    body { font-size: 14px; }
}

Aural, Screen, and Print Media Types

CSS can target different media types:

  • screen — for displays
  • print — for printers
  • aural — for speech synthesizers
